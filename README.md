# 🍽️ Restaurant Data Analysis using Web Scraping and Python

## 📌 Project Overview
This project focuses on extracting restaurant data from a website using web scraping techniques and performing exploratory data analysis (EDA) to uncover insights related to pricing, ratings, and cuisines.

---

## 🎯 Objective
- To collect real-time restaurant data using web scraping  
- To clean and preprocess the dataset  
- To analyze restaurant trends across different cities  
- To visualize insights using graphs and heatmaps  

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Selenium (for Web Scraping)  

---

## 🔍 Data Collection
- Extracted restaurant data using Selenium  
- Scraped multiple pages to gather large dataset  
- Collected features like:
  - Restaurant Name  
  - Price  
  - Rating  
  - Location  
  - Cuisine  

---

## 🧹 Data Cleaning
- Removed duplicates  
- Handled missing values  
- Converted price columns into numeric format  
- Standardized text data  

---

## 📊 Exploratory Data Analysis (EDA)
- Distribution of restaurant prices  
- Rating analysis  
- City-wise comparison  
- Cuisine trends  

---

## 📈 Visualizations
- Bar Charts  
- Histograms  
- Count Plots  
- Heatmap (correlation analysis)  

---

## 💡 Key Insights
- Some cities have higher average pricing compared to others  
- Highly rated restaurants are concentrated in specific locations  
- Certain cuisines dominate across multiple cities  
- Price and ratings show moderate correlation  

---

## 📌 Conclusion
This project demonstrates how web scraping and data analysis can be used to extract meaningful insights from real-world data. It highlights trends in restaurant pricing and customer preferences.
